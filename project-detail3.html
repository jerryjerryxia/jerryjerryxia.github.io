League of Legends Meta Analysis

As an avid League player, meta and balancing analysis has always interested me. This is exactly why I jumped at such a task the moment I became proficient with SQL and R.

Using MySQL, I constructed a database to host historic League data found on Kaggle. The dataset encompassed both detailed match history breakdowns and in-game balancing data such as champion stats, item stats,and so on.

I was responsible for delivery this entire project end-to-end: finding a dataset, defining the data schema, populating the database, running the analysis, all the way until presenting the results.

Some results were very surprising - for example, the champion with the highest overall stats at level 18 is not a tank, nor a bruiser, but a melee-mage!

Want to find out more? Feel free to check out the entire project on GitHub.
